**Blood & Organ Donation Services Platform**

TOHF-E-HAYAT is a secure, real-time web application designed to bridge the gap between voluntary donors and those in urgent need of blood or organ transplants. The platform focuses on user privacy while ensuring a seamless request-and-approval workflow.

## ‚ú® Key Features

- **Secure Authentication:** Users can create accounts and log in securely via Firebase Auth.
- **Privacy-First Workflow:** Donor contact numbers are hidden by default. Requesters must send a digital request, and only upon the donor's approval is the phone number revealed.
- **Real-Time Notifications:** A dedicated notification hub allows users to manage incoming requests (Approve/Deny) and track their own outgoing requests.
- **Full Profile Control:** Donors have the power to **Edit** their information at any time or **Delete** their profile if they are no longer available.
- **Medical Accuracy:** Mandatory blood type selection for all donors (Blood and Organ) to ensure medical compatibility.
- **Responsive Design:** Built with Tailwind CSS, the platform is fully optimized for both Desktop and Mobile devices.
